Lca¶

有向木の最小共通祖先(Lowest Common Ancestor)を求めます

Lca

メンバ¶

コンストラクタ¶

lca(int n)

\(n\) 頂点 \(0\) 辺の有向グラフを作ります.

制約¶

  • \(1 \le n \le 10^8\)

add_edge¶

void add_edge(int u, int v)

頂点 \(u\) から \(v\) への辺を追加します.

制約¶

  • \(0 \le u,v \lt n\)

operator()¶

int operator()(int u, int v)

頂点 \(u\) と \(v\) の最小共通祖先を返します.

\(u\) と \(v\) が互いに異なる木に所属する場合は \(-1\) を返します.

制約¶

  • \(0 \le u,v \lt n\)

  • このグラフは森である
