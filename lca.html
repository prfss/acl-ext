Lca¶

最小共通祖先(Lowest Common Ancestor)を求めるためのクラスです

Lca

制約¶

  • 入力は根から葉へ向きづけられた森である

メンバ¶

コンストラクタ¶

lca(int n)

節点数を \(n\) としてグラフを初期化します.

制約¶

  • \(1 \le n \le 10^8\)

add_edge¶

void add_edge(int u, int v)

節点 u から v へ枝を張ります.

制約¶

  • \(0 \le u,v \lt n\)

operator()¶

int operator()(int u, int v)

節点 u と v のLCAを返します.

u と v が互いに異なる木に所属する場合は \(-1\) を返します.

制約¶

  • \(0 \le u,v \lt n\)
