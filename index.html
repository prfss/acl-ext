acl-ext¶

AtCoderの問題で要求される(かもしれない)アルゴリズムを提供するライブラリです

特徴¶

  • 基本的には namespace aclext {...} の部分をコピペ(+適宜改造)して使う感じです

  • STL,ACLのヘッダーのインクルードが必要です

  • using namespace std が必要です

  • long long はドキュメントでは便宜上 ll と書かれていますが, コード中では long long と書いています
