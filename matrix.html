Matrix¶

和・差・積・累乗などの演算を備えた行列クラスです

Matrix<typename T>

\(T\) は要素の型です.

メンバ¶

コンストラクタ¶

matrix(int n, int m, T init = 0) // (1)
matrix(const vector<vector<T>> list) // (2)

  1. 要素を \(\text{init}\) で初期化した \(n \times m\) 行列を返します.

  2. \((i,j)\) 成分が \(\text{list}[i][j]\) であるような行列を返します.

制約¶

  1. \(1 \le n,m\)

    • \(\text{list}[i] (0 \le i \lt \text{list.size()})\) の長さはすべて等しい

    • リストは空ではない
