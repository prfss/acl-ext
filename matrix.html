Matrix¶

和・差・積・累乗などの演算を備えた行列クラスです

Verification

template <typename T, T(*add)(T,T), T(*mul)(T,T), T(*zero)(), T(*one)(), bool has_subtraction = false> Matrix

T は要素の型です. add, zero は加算とその単位元, mul, one は乗算とその単位元を表します. また has_subtraction == true の時, operator- を定義します.

制約¶

  • \(\mathrm{add}(x,\text{zero}()) = \mathrm{add}(\text{zero}(),x) = x\)

  • \(\mathrm{mul}(x,\text{one}()) = \mathrm{mul}(\text{one}(),x) = x\)

メンバ¶

コンストラクタ¶

matrix(int n, int m, T init = zero()) // (1)
matrix(const vector<vector<T>> list) // (2)

  1. 要素を \(\mathrm{init}\) で初期化した \(n \times m\) 行列を返します.

  2. \((i,j)\) 成分が \(\mathrm{list}[i][j]\) であるような行列を返します.

制約¶

  1. \(n,m \ge 1\)

    • \(\mathrm{list}[i] (0 \le i \lt \mathrm{list.size}())\) の長さはすべて等しい

    • リストは空ではない

identity¶

static Matrix<T> identity(int n)

\(n\) 次の単位行列 \(I_n\) , すなわち

\[\begin{split}(I_n)_{ij} = \begin{cases} \mathrm{one}() & (i = j) \\ \mathrm{zero}() & (\text{それ以外}) \\ \end{cases}\end{split}\]

を返します.
